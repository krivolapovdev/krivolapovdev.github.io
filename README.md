- **Frontend:** React + Vite
- **UI Framework:** Material UI (MUI)
- **Deployment:** GitHub Pages

## 📦 Getting Started

Clone the repo and run the development server:

```bash
git clone https://github.com/nell-shark/nell-shark.github.io
cd nell-shark.github.io
npm install
npm run dev
```
